hexo语法笔记

init

$ hexo init [folder]

新建一个网站，如果没有设置folder，则默认在当前文件夹

new

$ hexo new "title"

新建markdown格式文章

generate

$ hexo generate

生成静态文件，也可简写为：

$ hexo generate

server

$ hexo server

启动服务器，默认地址为：http://localhost:4000/，也可简写为：

$ hexo s

deploy

$ hexo deploy

部署之前的静态文件，也可以简写为：

$ hexo d

clean

$ hexo clean

清楚缓存文件(db.json)和已经生成的静态文件(public)，往往在发现对站点修改不生效时需要运行该命令。
